# GATE 2027 + Trading Checklist (Beginner)

A simple GitHub Pages checklist site for a complete beginner.

- Plan window: **Feb 21, 2026 to Jan 31, 2027**
- Fixed trading time: **9:00 AM to 3:00 PM**
- Goal: build GATE prep consistency while maintaining trading discipline

## Files
- `index.html` - layout and checklist sections
- `styles.css` - UI styling
- `script.js` - auto-generated weekly checklist + localStorage persistence

## Publish on GitHub Pages
1. Push repository to GitHub.
2. Go to repo `Settings > Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`, folder `/ (root)`.
5. Save. Site URL appears in Pages section.

## Notes
Checklist tick state is saved in your browser via localStorage.
